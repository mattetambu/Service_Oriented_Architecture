Get_image::inizialize_parameters ( string  path)
private

Inizializza l'istanza della classe Get_image (è chiamata da ogni costruttore).

Parametri
[in]pathDirectory di lavoro dove ricercare i files per creare la lista. Inizializza l'istanza della classe Get_image come segue:
  • genera un parametro in ingresso di tipo string che conterrà il nome dell'immagine da richiedere al server remoto
  • genera un parametro in uscita di tipo buffer per l'immagine da inviare al Client
  • crea la directory di lavoro del servizio se questa non esiste
