Documentazione delle funzioni membro

bool Horizontal_flip_image::execute ( )
virtual

Funzione che espleta il servizio di riflessione dell'immagine.

Restituisce
Risultato dell'esecuzione, true in caso di successo e false altrimenti. Il servizio di riflessione viene effettuato come segue:
  • genera nella directory di lavoro l'immagine da elaborare a partire dal parametro buffer in ingresso al servizio
  • vi applica l'operatore di riflessione ottenendo una nuova immagine (da ritornare al chiamante)
  • compila il parametro di uscita del servizio (di tipo buffer) con i dati dell'immagine riflessa
  • se REMOVE_IMAGES è impostato a true, rimuove le due immagini appena create (ormai inutili) dalla directory di lavoro

private

Inizializza l'istanza della classe Horizontal_flip_image (è chiamata da ogni costruttore).

Parametri
[in]pathDirectory di lavoro da assegnare al servizio dove salvare i files temporanei. Inizializza l'istanza della classe Horizontal_flip_image come segue:
  • genera un parametro in ingresso di tipo buffer atto a mantenere le informazioni relative all'immagine da elaborare
  • genera un parametro in uscita di tipo buffer per l'immagine riflessa
  • crea la directory di lavoro del servizio se questa non esiste
