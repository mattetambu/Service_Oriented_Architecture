Store_image::inizialize_parameters ( string  path)
private

Inizializza l'istanza della classe Store_image (è chiamata da ogni costruttore).

Parametri
[in]pathDirectory di lavoro da assegnare al servizio dove salvare i files temporanei. Inizializza l'istanza della classe Store_image come segue:
  • genera un parametro in ingresso di tipo string che conterrà il nome con cui memorizzare l'immagine sul Server
  • genera un parametro in ingresso di tipo buffer atto a mantenere le informazioni relative all'immagine da memorizzare
  • crea la directory di lavoro del servizio se questa non esiste
